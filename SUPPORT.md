# Getting Help

Before opening anything, search [existing issues] and [discussions].

## I have a question or want to discuss something

Open a thread in [Discussions]. Questions are not bug reports.

## I think I found a bug

Open an issue with the **Bug** template. Read [CONTRIBUTING.md] first;
issues from new contributors are auto-closed unless they have `lgtm`.

## I want to propose a feature

Open an issue with the **Feature** template, after checking that
something similar isn't already in Discussions.

## I found a security problem

Do not open a public issue. See [SECURITY.md].

## I want to chat with the maintainer

Don't. Open something on the tracker.
