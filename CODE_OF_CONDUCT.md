# Code of Conduct

This is a small project run by one person. The rules are short on purpose.

## Do

- Be useful. Open issues and PRs that respect other people's time.
- Disagree with the argument, not with the person. Bring evidence.
- Assume the other side wrote what they wrote in good faith.

## Don't

- Harassment, slurs, threats. Public or private. One strike, permanent ban.
- Brigading, doxxing, or weaponizing the tracker against anyone.
- Spam, agent-generated noise, recycled AI slop. See CONTRIBUTING.md.

## Scope

Applies anywhere this project's name is on the wrapper: issues, PRs,
discussions, commit messages, and any external chat about the project.

## Enforcement

The maintainer decides. There is no appeals process.

To report something privately, email **danielbusnz@gmail.com**. Include
links, screenshots, or whatever is concrete. Bad-faith reports count as
their own violation.

Maintainers are held to the same rules. If a maintainer is the problem,
report it the same way; another contributor will handle it.
